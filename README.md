# SCT_WD_02 - Stopwatch Web Application

## Objective

Build an interactive stopwatch web application using HTML, CSS, and JavaScript that allows users to accurately measure and record time intervals.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Start Stopwatch
* Pause Stopwatch
* Reset Stopwatch
* Lap Time Recording
* Responsive User Interface
* Accurate Time Tracking

## Project Workflow

1. UI Design
2. HTML Structure Development
3. CSS Styling
4. JavaScript Stopwatch Logic
5. Testing and Debugging
6. Deployment using GitHub Pages
